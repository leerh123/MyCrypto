# MyCrypto
## 介绍
大整数运算与部分密码学原语的简单实现。MyCyrpto简洁的接口可以让你快速上手，同时它还拥有优秀的性能。
## 功能

|     类别     | 具体操作                                  |
| ------------ | ----------------------------------------- |
| 大整数运算   | 加减乘除，移位运算，逻辑运算              |
| 大整数模运算 | 快速模幂，模逆                            |
| 生成随机数   | 生成大随机数，生成大素数                  |
| 对称密码     | AES                                       |
| 加密模式     | ECB，CBC，CFB，OFB，CTR                   |
| 公钥密码     | RSA                                       |
| 哈希函数     | MD5，SHA1，SHA256，SHA512，SHA3_256，SM3 |
| MAC          | HMAC,CMAC                                 |
